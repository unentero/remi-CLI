# remi-CLI
Remi-CLI is a CLI tool written in python, designed as a prototype for a semi-automatized studying method that uses Active Recall.
This is a very early version and I'm looking for feedback on bugs and features.
## Instalation
### EN
1. First you need to have installed python 3.14.4
``` bash
~$ python --version
Python 3.14.4
```
2. Install pip
#### Linux
``` bash
~$ pip --version
pip 25.1.1 from /usr/lib/python3.14/site-packages/pip (python 3.14)
```
#### Windows
``` powershell
C:> py pip --version
```
3. Clone the repo using git
4. Change to the repo directory
#### Linux
``` bash
~$ cd etc/remi-CLI
```
#### Windows
``` powershell
C:> cd /etc/remi-CLI
```
5. Install locally using pip
#### Linux
``` bash
~$ pip install --editable .
```
#### Windows
``` powershell
C:> py -m pip install .
```
6. Call the program using 'remi-cli' in your terminal
